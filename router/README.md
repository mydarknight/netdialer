Router Adapter
==============

use for different kinds of routers

general cmd format
------------------

```
// router mode
netdialer -r -u xxxxx@xxx.xy -p xxxxxx | hiwifi -wan -p xxxxx

// direct mode
hiwifi -p xxxxx | netdialer -u xxxxx@xxx.xy -p xxxxxx -ip stdin
```
